## Run Locally

Install dependencies

```bash
  npm install
```

Running Local

```bash
  npm run dev
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```


## Tech Stack

**Client:** React, Redux, Vite, Tailwind

**Database:** Mongo DB

**Server:** Node, Express

**Node Version:** Node v22.20.0
